# Gesture Control System 👋

A real-time gesture-based system to control system volume and brightness using Computer Vision.

## 🚀 Features

* Volume control using thumb–index distance (right hand)
* Brightness control using gesture mapping (left hand)
* Pinch gesture for mute
* Real-time hand tracking
* Dynamic on-screen UI feedback

## 🛠️ Tech Stack

* Python
* OpenCV
* MediaPipe

## ⚙️ How It Works

The system detects hand landmarks using MediaPipe and calculates the distance between fingers.
These values are mapped to system controls like volume and brightness in real time.

## 📌 Future Improvements

* Add more gesture controls
* Improve tracking stability
* Extend to applications like virtual mouse or gaming
